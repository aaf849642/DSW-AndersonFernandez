Etiquetas Semanticas:

HTML5 incorporo etiquetas con el fin de facilitar la estructura de la pagina web, permitiendo que esta sea mas logica y funcional, permitiendo simplificar lo conocido en HTML4.
Dentro de las etiquetas mas conocidas se encuentran: Header, Footer, Nav, Section, Aside, Article. Estos elementos incorporados en HTML5 evitan el exceso de etiquetas y se mejora su estructura para que esta se mas legible.
Adicionalmente se cuentan con las etiquetas semananticas de contenido multimedia, para insertar audio, video, graficos, asi como las usadas para facilitar el trabajo con formularios.
Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).

Listas y Enlaces:

Las listas en HTML permiten crear conjuntos de elementos en forma de lista dentro de una pagina, los cuales iran generalmente precedidos por un guion o un numero.
Los tipos de listas en HTML son:
1) Listas Ordenadas
Son las que permiten mostar elementos de la lista en orden.
2) Listas desordenadas
Son las que permiten mostar los elementos sin ningun tipo de orden, unicamente precedidos por una viñeta.
3) Listas de definiciones
Son ñas que nos sirven para montar listas en las que tenemos estructuras de valor y definición.
